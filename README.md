## Dificultades encontradas y cómo se resolvieron

Durante el desarrollo, una de las principales dificultades fue aprender nuevas tecnologías y herramientas como React, Express, SQLite, Git, GitHub, Render, Vercel y Cypress, además de organizar la aplicación utilizando componentes, props, estados y React Router para mantener una estructura clara y ordenada.

También surgieron inconvenientes al integrar el frontend con el backend, configurar CORS para el despliegue y lograr la persistencia del carrito mediante `localStorage`. Todos estos problemas se resolvieron mediante pruebas, depuración del código y una correcta configuración de la aplicación.
